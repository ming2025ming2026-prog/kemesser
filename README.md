# Kemesser Website

科默斯科技官网静态页面。

## GitHub Pages

仓库上传到 GitHub 后，可在：

Settings -> Pages -> Build and deployment

选择：

- Source: Deploy from a branch
- Branch: main
- Folder: /root

保存后，GitHub 会生成一个 Pages 访问地址。

## Files

- `index.html`: 首页
- `about.html`: 关于我们
- `thawing.html`: 复苏系列
- `cryo.html`: 冻存系列
- `cold-operation.html`: 冷冻操作
- `contact.html`: 联系我们
- `assets/`: 网站图片与视频资源

## Note

GitHub Pages 在国内访问稳定性不可控，更适合作为演示和代码托管。
